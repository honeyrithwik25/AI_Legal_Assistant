# Lexify — Free Legal Advice Platform

A vibrant, conversion-focused legal advice web app inspired by Vidhikarya. Built to compel visitors to sign up and use the platform.

## Features

- **Conversion-focused design** — Multiple sign-up CTAs, floating button, live ticker
- **Compelling hero** — Gradient text, stats, trust badges, clear value proposition
- **Sign Up / Log In** — Modal forms with smooth UX
- **Benefits section** — Why sign up: unlimited questions, notifications, history
- **Testimonials** — Social proof from Ravi, Priya, Amit
- **Browse by category** — Employment, Family, Civil, Criminal, Banking, etc.
- **Q&A cards** — Real questions with lawyer answers
- **Ask your question** — Form modal (can be gated by signup)
- **FAQ accordion** — Expandable answers
- **Responsive** — Mobile-friendly with collapsible menu

## How to Run

1. Open `index.html` in a browser, or
2. Use a local server: `npx serve .` or `python -m http.server`

## Tech Stack

- Vanilla HTML, CSS, JavaScript
- No build step or dependencies
- Fonts: Playfair Display, Outfit (Google Fonts)

## Design Highlights

- **Vibrant palette** — Gold/amber primary, emerald secondary, electric blue accents
- **Animations** — Glow effects, fade-in-up, hover micro-interactions
- **Trust elements** — "100K+ answered", "500+ lawyers", "4.9 rating", city names
- **Urgency** — "Sign up in 30 seconds", "No credit card", "24hr response"
